# GiphyXplorer
Simple iOS 🍎📱 browser 🔍 for https://giphy.com 👀🕺🛸

Search and share gifs.

<b> Requirements </b>
 - iOS 12.0 +
 - Swift 5
 - Xcode 10.2.1 +
